# Oasis Infobyte – Data Analyst Internship

## Tasks Overview



### Task 1 – EDA on Retail Sales Data
**Objective:** Perform Exploratory Data Analysis on retail sales data to uncover trends, patterns, and business insights.

**Tech Used:** Python, Pandas, Matplotlib, Seaborn

**Steps:**
- Loaded and cleaned the retail sales dataset
- Analyzed sales trends over time
- Identified top-performing products and categories
- Visualized revenue distribution and seasonal patterns
- Derived actionable business insights


### Task 2 – Customer Segmentation
**Objective:** Segment customers into distinct groups based on purchasing behavior using clustering techniques.

**Tech Used:** Python, Pandas, Scikit-learn, K-Means Clustering, Matplotlib, Seaborn

**Steps:**
- Loaded and preprocessed customer data
- Performed feature scaling
- Applied K-Means clustering
- Visualized clusters and interpreted each segment
- Provided business recommendations for each customer group



### Task 3 – Sentiment Analysis
**Objective:** Analyze text data to classify sentiments as positive, negative, or neutral.

**Tech Used:** Python, Pandas, NLTK, Scikit-learn, TF-IDF, Matplotlib

**Steps:**
- Loaded and cleaned the text dataset
- Applied text preprocessing (tokenization, stopword removal)
- Used TF-IDF for feature extraction
- Built a classification model and evaluated accuracy
- Visualized sentiment distribution



### Task 5 – Predicting House Prices with Linear Regression
**Objective:** Build a regression model to predict house prices based on features like location, size, and amenities.

**Tech Used:** Python, Pandas, Scikit-learn, Linear Regression, Matplotlib, Seaborn

**Steps:**
- Loaded and explored the housing dataset
- Handled missing values and encoded categorical features
- Built a Linear Regression model
- Evaluated using MAE, RMSE, and R² score
- Visualized actual vs predicted prices



### Task 6 – Wine Quality Prediction
**Objective:** Predict the quality of wine based on its chemical properties using machine learning.

**Tech Used:** Python, Pandas, Scikit-learn, Random Forest, Matplotlib, Seaborn

**Steps:**
- Loaded and explored the wine quality dataset
- Performed EDA and correlation analysis
- Built classification/regression models
- Tuned hyperparameters and evaluated model performance
- Visualized feature importance


### Task 7 – Fraud Detection
**Objective:** Detect fraudulent credit card transactions using machine learning techniques.

**Tech Used:** Python, Pandas, Scikit-learn, Logistic Regression, Random Forest, Matplotlib

**Steps:**
- Loaded and explored the imbalanced fraud dataset
- Handled class imbalance using SMOTE / undersampling
- Built classification models
- Evaluated using Precision, Recall, F1-Score, and ROC-AUC
- Visualized confusion matrix and ROC curve
